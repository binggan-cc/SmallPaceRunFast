## 实战工作流（作者亲测）

### Step 1：脚本规划
- Max 套餐每天只有 **4 个视频额度**，所有工作围绕这 4 个视频展开
- 先让 AI 编剧本、写分镜、写提示词
- 每个分镜/运镜至少准备 3-4 个可选项，用 LLM "抽卡"弥补质量缺陷

### Step 2：角色与场景（image-01）
- 每天 120 张图，每组 4 张 = 可做 30 组分镜场景
- **文生图抽卡**：生成 9 张选最优角色
- **图生图锁角色**：用 `subject_reference` 锁定角色一致性，生成不同场景的首帧图

```python
# 示例：角色参考生成场景
for shot in ["shot5", "shot7", "shot8", "shot9"]:
    image-01 I2I:
        subject_reference=[character_ref],
        n=4,
        aspect_ratio="16:9"
```

### Step 3：视频生成（Hailuo 2.3）

**关键技巧 1：能用图生视频，就不用文生视频**
- 文生视频无法保证角色/场景一致性
- 首帧参考图是场景一致性的延续

**关键技巧 2：6秒内多场景**
- 一个视频虽然只有 6s，但可以通过运镜和提示词变化拆分出多个短镜头
- 后期用剪辑软件拆分，提高"分镜产出率"

**关键技巧 3：运镜脚本要细**
- Hailuo 2.3 支持 15 种免费运镜指令
- 每组最多 3 个复合运镜
- 建议加 `[1s-2s]` 人工强制分镜切片

```python
hailuo_2.3_fast:
    first_frame_image=url,
    prompt=f"{scene_desc} [推进，变焦推进]",
    duration=6,
    resolution="768P"
```

**关键技巧 4：尾帧接续（ffmpeg）**
```python
# 生成视频
# 提取最后一帧作为下一个视频的首帧
extract_lastframe(f"{clip}.mp4", f"{clip}_lastframe.png")
```
- ⚠️ **限制**：跨视频不能做人脸分镜（第二个视频没有人物参考图，人物会变形/换人）
- ✅ 适合：纯场景、背影、空镜头

### Step 4：音频与合成
- Hailuo 2.3 视频默认**没有声音**
- BGM 技巧：把歌词换成"呢喃声/轻声低语"，比纯音乐氛围感更强
- 最后通过 ffmpeg 合并所有视频、音乐、TTS

## 资源最大化策略

### 策略1：图片批量抽卡
```python
payload = {
    "model": "image-01",
    "prompt": "...",
    "n": 9,  # 一次9张选最优
    "aspect_ratio": "16:9"
}
# 120张 = 13次API调用（9×13=117张）
```

### 策略2：角色一致性锁
```python
payload = {
    "model": "image-01",
    "prompt": "同一个人在地铁站...",
    "subject_reference": [{"type": "character", "image_file": "https://catbox.moe/xxx.png"}],
    "n": 4
}
```

### 策略3：15种免费运镜指令
- **移动**：左移、右移、推进、拉远、上升、下降
- **摇镜**：左摇、右摇、上摇、下摇
- **变焦**：变焦推近、变焦拉远
- **其他**：晃动、跟随、固定

## 三个弥补 AI 缺陷的创作心法

### 1. 镜头语言
- 少用多人同场景（AI 容易画崩）
- 多用**近景、人物特写、视线外放**（看向镜头外）
- 观众会自动脑补空间，减少 AI 出错的违和感

### 2. AI 特色：天然适合鬼片
- 鬼片本身就是抽象的、不讲逻辑的
- AI 的"废片"（画面故障、人物变形、夸张表情）在鬼片里反而成了氛围加成
- 作者原话："这些在正常片子里一眼假的内容，在鬼片里，是不是就符合逻辑了！天才！出院！"

### 3. 空镜头
- 空镜头（纯场景/黑屏）对一致性要求极低
- 适当空镜转场可以调动节奏、增加艺术感
- "没有了人物，一个纯场景的视频，它的废片率就大大降低了"
