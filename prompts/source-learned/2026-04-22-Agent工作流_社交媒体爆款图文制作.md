# 打破信息差：Agent工作流打造社交媒体爆款图文

## 核心成果

- 单图最高：**56万查看，1489收藏，336转推**
- 累计产出：少儿识字、菜谱图、安全知识图等**十几个爆款**

---

## 三步流水线

### 第一步：调研 — 模仿真掏钱的作业

**核心原则**：不模仿爆款（情绪梗/争议梗），要看**销量高**的图，掏钱买才是真需求

**操作流程**：
1. 打开小红书，按**销量排**
2. 把前几名的商品图拿下来
3. 扔进 AI Agent，让它**反推提示词**
4. 稍微改改就能直接用

> "我什么都没做，我只是从需求出发，找对了那个已经解决问题的答案"

---

### 第二步：生产内容 — 一次定制，批量出图

**核心方法**：从"写 Prompt"升级到**"写 Meta Prompt"**

```
Meta Prompt = 固定题干 + 填空变量
每次填不同答案 → 得到不同图
```

**示例**：
```
场景：[空格] + 字：[空格] + 风格：[空格] + 主题：[空格]
→ 填"健身房" → 健身卡
→ 填"厨房" → 厨房用品
```

**工具**：直接跟 CREAO Agent 对话，让它列出定制化选项，随机切换生成新提示词

---

### 第三步：自动化 — 你睡觉它上班

**核心操作**：把跑成功的对话**一键保存为 Agent**

**效果**：
- 明天、后天、下个月，随时打开，参数填一下，图就出来
- 加 **Scheduled Run**：按时间自动跑，不用打开应用
- 例如每天早上 7 点自动出一批新图，你睡醒只管挑

> "你不需要到场，它自己出产。你睡觉的时候它在工作，你出门的时候它在工作，你开会的时候它在工作！"

## 创业思维做社媒

### 核心逻辑

```
调研（挖痛点）→ 反推 Prompt → Meta Prompt（定制化）→ 保存 Agent + Scheduled Run（自动化）
```

### 关键洞察

| 思维 | 说明 |
|------|------|
| **需求导向** | 人的痛点无穷无尽 → 灵感无穷无尽 |
| **资产积累** | 每跑一次，库里加一件资产，越干越高效 |
| **一人公司** | 每条内容都是一次性劳动 → 很快做不下去；有这条链 → 沉淀成可复用资产 |

### 适用场景

- 社媒内容批量生产
- 信息收集 + 竞品分析
- 任何重复性工作 → 沉淀为 Agent

---

## 一句话总结

> **AI时代的工作方式：找到重复的事 → 沉淀成 Agent → Scheduled Run → 你睡觉它上班。**
