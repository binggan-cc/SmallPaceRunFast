# 平面几何字体海报生成器 — GPT Image 2 Prompt模板

**主题**：现代平面几何字体概念海报 / Geometric Typographic Concept Poster
**模型**：GPT Image 2
**时间**：2026-04-28

---

## 核心定义

这不是普通插画，也不是简单把文字放大后的字效海报。

**本质**：基于主题词含义自动构建**视觉隐喻**的现代平面设计海报。

```
文字（主视觉）
+ 几何图形/色块/线条/空间关系/透明叠层/模块化结构
= 语义延展 → 观众一眼感受到词语的气质/情绪/关系/内在含义
```

## 主题分类 → 视觉表达策略

### 1. 情绪型（孤独、自由、热烈、焦虑、松弛、压抑、浪漫、冷漠）
**表达重点**：色块关系、留白比例、图形距离、方向感、密度变化、视觉重心

### 2. 动作型（逃离、靠近、坠落、生长、爆发、旋转、流动、停滞）
**表达重点**：几何运动轨迹、字体切割与错位、方向线条、模块推进关系、斜向构图、速度感

### 3. 关系型（连接、对抗、平衡、依赖、分裂、共生、拉扯、融合）
**表达重点**：两/多主体之间的距离、重叠、分离、嵌套、包裹、对称、失衡、碰撞、交错、连接节点、中介结构

### 4. 抽象概念型（时间、秩序、边界、记忆、命运、欲望、真相、规则、偏见）
**表达重点**：找到简洁但具体的视觉隐喻 → 系统感排版、网格变化、遮挡、偏移、层级结构、模块重复与变化

### 5. 具象物象型（火焰、种子、月亮、桥梁、岛屿、风、镜子）
**表达重点**：提炼为符号化、几何化、排版化、结构化的视觉表达——**不要写实插图**

## 构图原则

```
画面元素数量控制：
├── 1组核心文字主体
├── 1组核心几何隐喻系统
└── 少量辅助线条/标签/点阵/小字信息
```

- **必须保留足够留白**，让画面透气、清爽、有当代感
- 文字可以：**横排/竖排/拆分/错位/重组/局部遮挡/嵌套/切片/偏移/透明覆盖**
- 文字与几何图形可形成：**正负形关系**
- 图形元素可以：**穿过/支撑/切割/包裹/与文字形成结构互动**
- **构图可以大胆，但文字可读性不能丢失**

## 视觉风格方向

```
modern graphic design
geometric composition
typographic poster
experimental typography
clean flat design
crisp vector-like precision
contemporary visual identity aesthetics
```

### 气质要求
- 现代、平面、清亮、理性
- 有设计感、有实验性、有系统感、有高级感
- **不复古、不厚重、不脏乱、不空洞**

### ❌ 避免风格
- 普通插画 / 写实摄影 / 复古文学封面
- 包豪斯旧海报感 / 构成主义老派风格
- 厚重建筑隐喻 / 暗黑戏剧化
- 电商海报感 / PPT封面感 / 过度模板感

## 配色方案（按主题自动匹配）

| 主题 | 配色 |
|------|------|
| 活力/热烈/爆发 | bright orange、coral red、vivid yellow、electric blue |
| 冷静/秩序/理性 | navy、black、white、gray、silver blue |
| 温柔/记忆/柔和 | soft pink、warm white、pale green、light yellow、muted lavender |
| 冲突/危险/对抗 | red、black、white、neon green（高对比）|
| 自由/轻盈/开放 | sky blue、mint green、white、bright yellow、aqua cyan |
| 神秘/哲思/深层意识 | indigo、deep violet、black + 亮色点缀 |

### 质感要求
- 可以有：极轻微渐变、少量透明叠层、细腻克制的数字感
- **不要**：做旧、脏、厚重、灰暗沉闷
- **整体要**：轻盈但有力量，现代但不空洞

---

## 字体与排版

- 中文：无衬线标题字、几何字体、现代设计感字体
- 英文：bold grotesk、condensed sans、geometric sans、editorial sans
- 允许局部变形，但**不能影响整体识别**
- 字体与图形必须有**明确互动关系**（不是图文分离）
- 可加入一句非常短的副标题（英文如 "A visual state of ___"），但**必须克制**，不抢核心文字视觉权重
