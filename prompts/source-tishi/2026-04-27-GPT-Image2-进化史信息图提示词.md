## 完整提示词（19原则精炼版）

> 你要生成的不是普通插画，也不是把信息机械排成一列的时间轴，而是一张"高完成度、强组织性、强阅读价值、兼具知识含量与高级审美的进程演化信息图海报"。

### 核心能力
用户只输入一个主题 → 自动延展出完整阶段结构、关键节点、代表性视觉对象、时间分段、核心说明与整体叙事逻辑

### 适用范围
文明史 / 产品演化 / 技术发展 / 朝代更替 / 帝王谱系 / 物种进化 / 器物演变 / 服饰演变 / 饮食演变 / 品牌发展 / 形态变迁 / 制度发展 / 行业阶段性变化

## 19条创作原则速查

| # | 原则 | 核心要点 |
|---|------|----------|
| 1 | 先理解主题，再决定结构 | 历史→年代推进 / 产品→形态变化 / 生物→进化链条 / 抽象→发展阶段 |
| 2 | 画面是被空间化的进程 | 递进式台阶/阶梯/斜向路径/展陈式轨迹，不用简单时间轴 |
| 3 | 兼具知识性与审美性 | 有内容密度、有设计美感，不是装饰海报 |
| 4 | 高级克制优雅专业 | 参考高端展览信息设计/博物馆展板/视觉百科，不要网红化 |
| 5 | 视觉方案因题而变 | 年代vs结构变化 / 实物vs人物 / 考古感vs科技感 |
| 6 | 配色跟随主题，不许固定 | 因主题语义/情绪/时代/材料/文化决定，不许默认暖棕米白 |
| 7 | 深浅模式判断逻辑 | 考古/教育/器物→浅色；科技/宇宙/赛博→深色 |
| 8 | 建立主题驱动的配色系统 | 背景/结构/标题/文字/标签/强调/注释 系统化配色 |
| 9 | 字体克制创新 | 断行/错位/叠压/顺应路径排布，但不失清晰度 |
| 10 | 字体匹配主题气质 | 历史→文化碑帖感 / 科技→利落精密 / 时尚→编辑化 |
| 11 | 版式结构强 | 主标题 + 副标题(进程逻辑) + 主体路径 + 阶段节点 + 注释系统 + 底部收束 |
| 12 | 时间与阶段清楚 | 年份/世纪/阶段编号；5-12个阶段按复杂度浮动 |
| 13 | 每个阶段代表性视觉对象 | 器物/产品/人物/服饰形态/化石/标本，像展品陈列 |
| 14 | 文本少而准 | 阶段标题+时间标签+一句话说明，不堆砌 |
| 15 | 图文关系真正融合 | 文字/对象/引线/标签共同组成完整知识可视化系统 |
| 16 | 阅读体验层层递进 | 主标题→进程路径→阶段对象→说明→细节标签 |
| 17 | 重点避免问题 | 不普通PPT感 / 不廉价百科图 / 不米白棕色模板 / 不花哨 |
| 18 | 智能细化内容 | 判断起点终点+阶段+对象+配色+字体+信息密度 |
| 19 | 最终输出感觉 | 高端+优雅+知识含量+图文深度融合+因主题而异 |

## 关键设计公式

```
视觉结构 = 起点 + 递进式台阶/路径 + 终点
阶段数 = 5-12个（按复杂度智能调整）
配色 = 因主题建立专属系统（背景/结构/标题/标签/强调）
阅读路径 = 主标题 → 进程路径 → 阶段对象 → 说明 → 细节标签
```
