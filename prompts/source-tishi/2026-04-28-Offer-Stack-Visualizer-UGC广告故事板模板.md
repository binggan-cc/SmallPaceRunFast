## 核心概念

**Offer Stack Visualizer** = 把一个商业offer拆解成6个不同角度的UGC广告概念，一次性生成完整的故事板包。

每个广告概念包含：
- Hook句 / 开场镜头 / 创作者人设 / 核心信息 / 场景递进 / 产品露出 / CTA / 情感调性 / 转化理由

## Prompt 模板结构

### Campaign Details（基础信息）
```
- Brand / business name: [NAME]
- Product name: [PRODUCT]
- Industry: [INDUSTRY]
- Product type: [PHYSICAL PRODUCT / APP / SERVICE / SUBSCRIPTION / COURSE]
- Target audience: [AUDIENCE]
- Main problem solved: [PROBLEM]
- Main promise: [BENEFIT / TRANSFORMATION]
- Platform focus: [TIKTOK / INSTAGRAM REELS / META ADS / YOUTUBE SHORTS]
- Positioning: [PREMIUM / MODERN / PLAYFUL / HIGH-PERFORMANCE / TRUSTWORTHY / BOLD]
```

### Storyboard 6大概念框架
| # | 角度 | 目的 |
|---|------|------|
| 1 | Problem-first hook | 痛点共鸣，吸引注意力 |
| 2 | Transformation / Before-After | 效果承诺，激发向往 |
| 3 | Testimonial / Social proof | 信任背书，降低顾虑 |
| 4 | Demo / Product-in-use | 真实展示，建立认知 |
| 5 | Objection-handling | 消除疑虑，推动决策 |
| 6 | Urgency / Offer-driven | 限时压力，催促行动 |

## 4个案例速览

### Case 1 — VELORA Sleep Spray
- **品类**：睡眠喷雾（sleep & wellness）
- **受众**：压力大的职场人、浅眠者、25-45岁注重健康的男女

- **核心钩子方向**：睡前仪式感、自然放松、摆脱失眠焦虑

- **核心钩子方向**：肌肤焕亮、简单高端仪式、每日可见的效果

### Case 3 — TaskMint AI Planner

## 核心设计原则

**做加法 vs 做减法**
- ✅ 加：清晰分隔、脚本片段、场景面板、CTA标签、创意注释
- ❌ 减：混乱布局、模糊概念、过量文字、随机审美

**一张图讲清楚整个Campaign策略**
- 每个Concept都是独立的测试变量
- 6个角度覆盖用户决策全链路：注意 → 兴趣 → 信任 → 认知 → 消除疑虑 → 行动
