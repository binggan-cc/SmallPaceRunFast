# GPT Image 2 · 字体美学 + 文字图鉴 + 大字报系列

> 核心定位：基于词义深度理解的视觉转译高级概念海报
> 不是"把字画出来"，不是"带说明文字的插画"，而是将词语的**精神状态、命运质感、情绪张力与象征意味**转译为展览级视觉海报

---

## 版本一：标准版（图文均衡）

### 核心原则

**整张海报必须是一个完整场域**
- 不做割裂的多栏信息板
- 文字、图像、留白共同存在于同一个视觉场域
- 阅读路径像"漫游"和"流动"，不像看表格

### 五大机制

| 机制 | 要点 |
|---|---|
| **深度语义理解** | 表层含义 + 深层寓意 + 文化联想 + 人物命运感 + 隐性冲突 + 象征性 |
| **图文融合** | 文字"长"在画面里，与图像彼此咬合（站在字前 / 藏入字内 / 穿过字形 / 借字留白构成动作） |
| **阅读层次** | 巨大主标题 → 核心视觉隐喻 → 少量阅读信息 → 辅助内容 |
| **视觉隐喻** | 单一而强，与词义高度绑定（尺度反差 / 柔软与坚硬并置 / 秩序与崩塌冲突） |
| **色彩逻辑** | 2~4种主色，高级克制，拒绝廉价渐变和俗艳拼色 |

### 构图公式

```
巨型文字骨架 + 简洁承载面 + 精准角色/物体演绎 + 巧妙补充元素 + 流动式阅读
```

### 辅助文字规则

- 必须短、准、巧、轻
- 像展签、注解、诗性补白，不做信息栏
- 禁止：假编号 / 假坐标 / 随机英文碎片 / 无关排版噪音

### 禁用项

- 图文分家 / 信息分栏割裂 / 廉价模板感 / 拥挤花哨

## 版本二：图形高存在感版

### 与标准版的差异

| 维度 | 标准版 | 图形高存在感版 |
|---|---|---|
| **图像比例** | 图文均衡 | 图形元素占更大比例 |
| **主体尺度** | 克制 | 主体更大、更突出 |
| **文字密度** | 较低 | 更低，文字更精简 |
| **视觉冲击力** | 强 | 极强，图像主导 |
