# Apple Style 自然科普海报生成系统

- **主题：** Apple风海报生成 / 自然科普视觉 / 物种信息图
- **来源：** 用户自定义提示词模板
- **时间：** 2026-04-27

---

## 整体视觉方向

- **风格：** Apple keynote 风格 — 极简、纯白、干净、现代、高级感
- **比例：** 9:16 竖版
- **关键词：** Apple-inspired, premium editorial, pure white background, hero subject, clean typography, minimal infographic, high-end science poster

---

## 画面结构

### 顶部左侧标题区
```
中文大标题：{中文物种名}
中文副标题：{一句有吸引力的物种定位}
细短横线
英文名：{英文物种名}
分布信息：主要分布：{分布区域}
```

### 中部与下中部主体视觉
- 主体占据画面 **50%-70%** 视觉面积
- 超高清、真实、强烈立体感
- 白色背景，不添加复杂自然环境
- 可保留少量简洁承托物（树枝/岩石/雪地/沙土/木皮）
- 真实阴影，高级产品摄影感

### 底部信息区（四列极简信息栏）
- 细线 icon + 彩色小标题 + 1-3行短说明
- 极细浅灰竖线分隔
- **不使用** 卡片框、圆角背景、大面积色块

| 栏目 | 内容 |
|------|------|
| 1 | {重点特征1标题} + {短说明} |
| 2 | {重点特征2标题} + {短说明} |
| 3 | {重点特征3标题} + {短说明} |
| 4 | {重点特征4标题} + {短说明} |

### 底部总结句
- 灰色小字，居中
- 高级、克制、有记忆点的科普总结

---

## 字体与排版

| 元素 | 规格 |
|------|------|
| 中文大标题 | 大号、黑色、稳重有力量 |
| 副标题 | 灰色、中等字号、字距略宽 |
| 英文名 | 小号灰色、简洁现代 |
| 正文 | 清晰现代中文、保持可读 |
| 原则 | 所有文字必须留有足够呼吸感 |

---

## 色彩规范

| 元素 | 颜色 |
|------|------|
| 背景 | 纯白 / 极浅灰 / 轻微柔光渐变 |
| 主标题 | 黑色或深石墨色 |
| 副标题与正文 | 中性灰 |
| 强调色（仅用于 icon 和小标题） | 暖棕 / 冷蓝 / 松石绿 / 紫色 / 橙色 |

> 颜色只用于 icon 和小标题，**不大面积铺色**

---

## 禁止项

- ❌ 淡黄色旧纸背景
- ❌ 复杂信息图网格
- ❌ 圆角卡片
- ❌ 厚边框
- ❌ 大面积装饰图形
- ❌ 无关 logo / 多余小字
- ❌ 主体太小
- ❌ 文字压住主体
- ❌ 底部信息区过度拥挤
- ❌ 儿童科普风 / 卡通风 / 低端展板风

---

## 图像质量要求

- 2K 高清质感，细节清晰，主体锐利
- 真实光影：毛发 / 鳞片 / 甲壳 / 皮肤褶皱 / 羽毛 / 斑纹
- 避免：变形 / 错误肢体 / 错误解剖结构 / 模糊主体 / 低质贴图 / 塑料感 / 卡通感

---

## 核心设计公式

```
极简标题区（顶部）
    ↓
超大主体占画面 50-70%（中部）
    ↓
四列icon+标题+短文（底部）
    ↓
一句总结收尾
```
