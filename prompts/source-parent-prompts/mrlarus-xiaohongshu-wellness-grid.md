## Original Prompt

```text
请生成一张适合小红书发布的“养生赛道”图文内容母版大图。

【主题规则】
本次主题为：【这里写一句主题内容】。
如果【主题】填写“随机”，请自动选择一个适合小红书传播、适合收藏和涨粉的养生主题。可优先从这些方向中自由选择并发挥：
熬夜修复、睡眠改善、脾胃养护、早餐养生、上班族轻养生、手脚冰凉、气色调理、久坐舒缓、泡脚养生、换季养护、情绪舒缓、女性日常养护、秋冬滋养、春季养护、低成本养生习惯。

【画布与排版要求】
- 整体画布比例为 3:4，竖版构图，符合小红书图片阅读习惯。
- 画面内部为 2×2 四宫格排版，共 4 张可独立裁切的小图。
- 四张小图之间要有清晰分隔、留白或卡片边界，方便后期切图。
- 每张小图都必须构图完整、内容独立、可单独发布。
- 每张小图中的主体与文字不要贴边，避免后期裁切受影响。

【核心风格要求】
整体风格必须符合“小红书优质养生博主内容”的审美方向：
- 清爽、温和、自然、干净、舒服、生活化、轻专业
- 有收藏感、阅读感、可信感，不要像广告图，也不要像医院宣传页
- 更像高质量博主整理出的实用养生笔记，而不是营销海报
- 画面精致、有统一感，但不要僵硬模板化
- 适合年轻人、上班族、女生用户浏览和保存

【风格自由度要求】
请不要死板地固定一种配色或同一种元素。
请先理解【主题】的内容气质，再自动匹配最合适的视觉风格、主色调、辅助色、生活元素和氛围表达，但整体仍需保持在“小红书养生卡片风”的统一大方向中。
也就是说：
- 如果主题偏“熬夜 / 睡眠 / 修复”，可偏米白、奶油色、浅绿、柔和暖光、夜晚与晨间元素
- 如果主题偏“脾胃 / 早餐 / 温补”，可偏燕麦色、浅杏色、暖米色、食物与餐桌感
- 如果主题偏“清爽 / 去火 / 夏季 / 情绪舒缓”，可偏浅绿、浅蓝、白色、清透轻盈感
- 如果主题偏“女性养护 / 气色 / 经期前后”，可偏柔粉、豆沙、杏粉、米白等温柔色系
- 如果主题偏“秋冬滋养 / 手脚冰凉”，可偏暖棕、奶咖、米杏、柔和暖调
- 允许根据主题加入最自然的生活化道具、食材、植物、器皿、便签、清单、插画、人物场景等
- 允许轻微变化和创意发挥，但不要偏离“小红书养生内容卡片”的整体调性

【视觉语言要求】
请使用高完成度、平台友好的视觉表达：
- 可采用“生活方式海报 + 轻信息卡片 + 小红书笔记封面”的融合风格
- 可使用：圆角卡片、纸张感、便签感、柔和自然光、轻杂志排版、温和插画、生活感实物、干净背景
- 适合搭配的元素包括：温水、茶饮、早餐、谷物、水果、植物、日常器皿、作息图标、轻插画人物、手写感小装饰、贴纸感提示语等
- 整体要有呼吸感和留白，不要堆满，不要低质廉价
- 文字清晰可读，中文排版整齐，适合手机端阅读
- 每张图的信息量适中，不要出现过密小字

【四张图的固定内容结构】
请保持四张图为同一主题下的系列内容，并按以下结构输出：
1）第1张：封面图
- 用于吸引点击
- 必须有醒目的中文大标题
- 可加入一句副标题，点明人群、场景或收益点
- 视觉上最抓人，最有“小红书封面感”
- 标题应口语化、生活化、好理解，有收藏和点击欲望

2）第2张：共鸣图 / 自查图
- 用于让用户迅速代入
- 列出与【主题】相关的 3～5 个常见表现、误区、身体信号或生活状态
- 让用户产生“这说的就是我”的感觉
- 表达要简洁，不说教，不夸张

3）第3张：方法图 / 建议图
- 输出与【主题】相关的 3～5 条日常养生建议
- 内容必须简单、温和、可执行、低门槛
- 更偏“生活方式建议”，不要像临床诊断
- 可以加入作息、饮食、运动、休息、情绪、节奏调整等内容

4）第4张：总结图 / 清单图
- 用于促进收藏与转发
- 用一句话总结主题核心逻辑
- 再整理成一份简洁明了的行动清单 / 修复清单 / 日常提醒
- 要有“建议收藏”的感觉，但不要过度营销

【文案要求】
- 所有文字必须为中文
- 标题尽量简洁有力，适合小红书表达习惯
- 文案整体口语化、自然、真诚、轻提醒感
- 适合使用“你是不是也这样”“建议收藏”“普通人也能做到”“先别急着补”“先从这几步开始”等自然表达
- 不要写成长篇大论，不要满屏大段文字
- 每张图聚焦一个重点，利于扫读

【平台与审核友好要求】
内容必须符合小红书风格与审核友好方向：
- 聚焦日常作息、饮食、生活习惯、情绪调节、轻养护建议
- 不要涉及疾病诊断
- 不要出现明确治疗承诺
- 不要出现“根治、速效、包好、立刻见效、药到病除”等夸张疗效表述
- 不要有保健品硬广感
- 不要制造焦虑，也不要神化某个方法
- 整体应传达“温和改善、节奏修复、长期养护”的理念

【最终效果目标】
请生成一张高完成度的小红书养生图文母版：
- 一张大图中包含 4 张竖版小图
- 四张图风格统一、但细节有变化
- 每张图都可以单独裁切使用
- 看起来像真实优质小红书博主发布的系列养生内容
- 具备较强的收藏感、阅读感、分享感和关注转化潜力
```
