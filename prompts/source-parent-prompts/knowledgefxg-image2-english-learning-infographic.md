## Original Prompt

用Image-2测试了个非常不错的英语学习教育图 Prompt 模板，可以套用在各个领域。

提示词：

一张干净的英语学习信息图海报，标题为“20种常见蔬菜”，采用整齐的网格布局（4x5），每个卡片展示一种蔬菜。每个卡片包含：

- 写实风格的蔬菜图片
- 英文名称（加粗）
- 英式或美式音标（IPA）
- 简短英文描述

整体风格：

- 极简设计
- 白色背景
- 柔和的绿色和浅色系配色
- 卡片式 UI 设计
- 圆角边框
- 排版整齐
- 视觉平衡
- 适合英语初学者学习
- 类似课堂教学海报
- 适合社交媒体传播
- 高分辨率
- 4K
- 信息图风格

尺寸：1080x1350

## Reusable Template

```text
Create a clean educational infographic poster titled "{TITLE}".
Use a neat grid layout ({GRID}), with each card showing one {ITEM_TYPE}.

Each card must include:
- A realistic image of the item
- The English name in bold
- British or American IPA pronunciation
- A short English description

Overall style:
- Minimalist design
- White background
- Soft green and light color palette
- Card-based UI design
- Rounded borders
- Clean typography and balanced spacing
- Suitable for beginners
- Classroom-poster feel
- Social-media-friendly infographic
- High resolution, 4K look

Size: 1080x1350
```
